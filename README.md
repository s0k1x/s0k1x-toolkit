## 📋 Requirements

|-----------|---------|
| **OS** | Kali Linux 2024.x / 2025.x (Rolling) |

| **Shell** | Bash 5+ |
| **Python** | 3.11+ |
| **Git** | Latest |
| **Internet** | Required during installation |

## 📥 Clone the Repository
```bash
git clone https://github.com/s0k1x/s0k1x-toolkit.git
```
